# 🧪 First Lab Practice

This directory contains practice codes for basic searching and sorting algorithms.

## 📝 Problem Statements

### 1. Array Linear Search (`Array linear search.cpp`)
**Question:** Write a C++ program to input an array of `n` elements and a `key`. Use **Linear Search** to find if the `key` exists in the array. If found, print its index; otherwise, print "Not Found".

### 2. Array Binary Search (`Array Binary Search.cpp` & `Binary Search.cpp`)
**Question:** Write a C++ program to input an array of `n` elements and a `target` element. Sort the array (if not already sorted) and use **Binary Search** to find if the `target` exists. Print whether the element was found or not.

### 3. Insertion Sort (`Insertion Sort.cpp`)
**Question:** Write a C++ program to sort an array of `n` integers in ascending order using the **Insertion Sort** algorithm.
