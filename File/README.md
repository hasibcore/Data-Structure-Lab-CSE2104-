# 📁 File Handling

## 📝 Problem Statement

### File Read and Write Operations (`File_Read_Write.cpp`)
**Question:** Write a C++ program to demonstrate basic file handling operations. The program should take a string input from the user, write it to a text file (`data.txt`), and then read the content back from the file to display it on the console.

**Description:**
The program uses `ofstream` to create and write to the file, and `ifstream` to open and read from the file line by line.
