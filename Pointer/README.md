# 📌 Pointer

## 📝 Problem Statement

### Swapping Numbers using Pointers (`Swap.cpp`)
**Question:** Write a C++ program to take two numbers as input from the user and swap their values using pointers and a custom `swap` function.

**Description:**
The program defines a `swap` function that takes two integer pointers as arguments. By passing the memory addresses of the variables (`&num1`, `&num2`), the function modifies the original values directly in the memory.
