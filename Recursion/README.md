# 🔄 Recursion

## 📝 Problem Statement

### Factorial using Recursion (`Factorial.cpp`)
**Question:** Write a C++ program to find the factorial of a given positive integer using a recursive function.

**Description:**
The program takes an integer $n$ as input and calculates its factorial recursively using the relation:
`factorial(n) = n * factorial(n-1)`
with the base case `factorial(1) = 1` and `factorial(0) = 1`.
