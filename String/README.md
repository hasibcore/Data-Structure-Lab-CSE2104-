# 🔤 String

## 📝 Problem Statement

### Palindrome Checker (`Palindrome.cpp`)
**Question:** Write a C++ program to check if a given string is a palindrome. A palindrome is a word, phrase, or sequence that reads the same backwards as forwards (e.g., "madam").

**Description:**
The program takes a single word as a string input and uses a two-pointer approach (one pointer at the beginning and one at the end) to compare the characters. If all corresponding characters match until the pointers meet, the string is a palindrome.
